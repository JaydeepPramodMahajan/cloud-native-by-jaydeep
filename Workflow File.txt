Its my Devops pipeline for the End-to-End CI/CD Pipeline for a Python Web Application using GitHub Actions, Docker, Kubernetes, Helm, and AWS

Description:- 
The application is built using FastAPI (Python) and provides REST APIs for managing tasks. Every code change pushed to GitHub automatically triggers a GitHub Actions workflow that performs code quality checks, runs tests, builds a Docker image, pushes it to Docker Hub, and deploys the latest version to a Kubernetes cluster using Helm. The application is monitored using Prometheus and Grafana to provide insights into application health and performance.
